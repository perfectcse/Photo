# Photo Gallery Web App

This project is a React-based Photo Gallery application built using **React, Vite, and Tailwind CSS**. The application fetches photos from a public API and displays them in a responsive grid layout.

## Features

- Fetch photos from the **Picsum Photos API**
- Display photos in a **responsive grid layout**
- Search photos by **author name**
- Mark photos as **favourites**
- Save favourites using **localStorage**
- Optimized performance using **useMemo and useCallback**

## Technologies Used

- React
- Vite
- Tailwind CSS
- JavaScript (ES6)
- LocalStorage API
